# ScholarNet

## Overview

ScholarNet is an AI-driven learning platform designed to enhance the educational experience for elementary, high school, and college students. The platform utilizes React.js for frontend development, MongoDB for backend support, and Node.js with Express.js for server-side operations. ScholarNet offers a range of features including live sessions, course uploads, AI-based content moderation, and parental controls to ensure a safe and interactive learning environment.

## Features

- **AI-Driven Content Moderation**: Automated monitoring and moderation of content to maintain a safe learning environment.
- **Live Sessions**: Real-time interaction between students and educators.
- **Course Uploads**: Seamless upload and management of educational materials.
- **Parental Controls**: Tools for parents to monitor and control their children's learning activities.
- **Interactive Learning**: Engaging tools and resources to enhance the learning experience.

## Technology Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express.js
- **Database**: MongoDB
- **APIs**: Fetch API for data retrieval
- **AI**: AI-based content moderation

## Installation

### Prerequisites

- Node.js
- MongoDB

### Steps to Install

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mozeb-Ahmed-Khan/scholarnet.git
2. **Navigate to the project directory:**
   ```bash
   cd scholarnet
3. **Install backend dependencies:**
   ```bash
   cd backend
   npm install
4. **Install frontend dependencies:**
   ```bash
   cd ../frontend
   npm install
5. **Set up the database**: Ensure MongoDB is running and create a database for ScholarNet.
6. **Start the backend server**:
    ```bash
    cd ../backend
    npm start
7. **Start the frontend server**:
    ```bash
    cd ../frontend
    npm start

### Usage

1. **Access the Platform:** Open your browser and navigate to http://localhost:3000 to access the frontend application.
   
2. **Register and Log In:** Use the registration and login features to access the platform.

3. **Explore Features:** Utilize the live sessions, course uploads, and parental controls as needed.
